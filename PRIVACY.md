# Privacy

WeaveNet for Copilot 不收集遥测数据，也不运营独立的数据收集服务。

## 本地数据

API Key 使用 VS Code SecretStorage 保存，并按扩展为每个连接生成的稳定 UUID 隔离。UUID 是普通配置身份，不是秘密。通过扩展命令删除连接时，对应 API Key 也会被删除。扩展设置由 VS Code 配置系统管理，其中包括自定义 `requestHeaders` 的名称和值；这些请求头不受 SecretStorage 保护，因此不应包含 API Key、令牌或其他敏感信息。调试日志不会记录 API Key 或提示词正文。

## 请求数据

当你使用模型时，对话、代码、图片、工具定义和工具调用结果会发送到你配置的 sub2api 中转站，并可能继续发送到该中转站配置的上游模型服务。数据处理、日志保留和计费规则由中转站及上游服务决定。

用户应自行确认所使用服务的隐私政策和安全要求，不应向不受信任的服务发送机密信息。
